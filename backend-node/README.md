# LitRevTool Node.js Backend - Complete Implementation

**Full-featured Node.js backend** - 100% feature parity with Python, reduced server load.

## Quick Start

\`\`\`bash
# Install dependencies
npm install

# Install Playwright browsers
npx playwright install chromium

# Build TypeScript
npm run build

# Run in development mode
npm run dev
\`\`\`

## Complete Feature Set

- ✅ Multi-Strategy Web Scraping (HTTP + Playwright with automatic failover)
- ✅ Google Gemini AI Integration (semantic filtering + LaTeX generation)
